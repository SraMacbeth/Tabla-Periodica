# 🧪 Tabla Periódica de los Elementos - CSS Grid Challenge

Una representación visual detallada de la tabla periódica de los elementos, construida exclusivamente con tecnologías web fundamentales. Este proyecto demuestra habilidades avanzadas en maquetación bidimensional y organización de estructuras complejas en el frontend.

## 📋 Descripción

El objetivo de este proyecto fue recrear la estructura técnica de la tabla periódica utilizando un sistema de rejilla (**Grid Layout**). Cada elemento químico está representado con su número atómico, masa atómica, símbolo y nombre, organizados mediante áreas de grid específicas para mantener la coherencia visual.

## 🚀 Funcionalidades y Detalles Técnicos

* **CSS Grid Layout:** Uso intensivo de `grid-template-columns` y `grid-template-rows` para crear la matriz de 19x10 que conforma la tabla.
* **Grid Areas:** Cada celda de elemento utiliza `grid-template-areas` para posicionar internamente los datos (número atómico, símbolo, etc.).
* **Categorización por Color:** Los elementos están diferenciados visualmente por familias (metales, no metales, gases nobles, actínidos, lantánidos, etc.) mediante clases de CSS.
* **Diseño Limpio:** Uso de tipografías modernas y una paleta de colores equilibrada para facilitar la lectura de datos científicos.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica para los 118 elementos y secciones informativas.
* **CSS3:** Maquetación avanzada, sistemas de colores y gestión de espaciados. No se utilizaron frameworks externos, resaltando el dominio de CSS puro (Vanilla CSS).
